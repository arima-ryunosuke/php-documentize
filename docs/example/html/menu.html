Namespace

  • N (global)
    • I Ginterface グローバルに定義されたインターフェースです
    • T Gtrait グローバルに定義されたトレイトです
    • C Gclass グローバルに定義されたクラスです
    • C GLOBAL_CONSTANT
    • F global_function グローバルに定義された関数です
  • N NS
    • N NSS
      • C C これはネストされた名前空間用です。実質的な意味はありません
      • F fff これはネストされた名前空間用です。実質的な意味はありません
    • I Ninterface 名前空間に定義されたインターフェースです
    • T Ntrait 名前空間に定義されたトレイトです
    • C Nclass 名前空間に定義されたクラスです
    • C Mclass マジック/プロパティメソッドのサンプル用クラスです
    • C Rclass これは継承関係を説明するためのクラスです
    • C Tclass 色々なタグを試すためのクラスです
    • C NAMESPACE_CONSTANT
    • F namespace_function 名前空間に定義された関数です
    • F package_function 色々なタグを試すための名前空間関数です
