    • C Gclass (3) グローバルに定義されたクラスです

  • N NS (10) サンプルドキュメント生成用の叩き台です（名前空間）

      • C C (1) これはネストされた名前空間用です。実質的な意味はありません
      • F fff これはネストされた名前空間用です。実質的な意味はありません

    • C Tclass (5) 色々なタグを試すためのクラスです
    • C NAMESPACE_CONSTANT 名前空間の定数です
