Structure (Symfony\Component\Console\Input\ArrayInput)

class Symfony\Component\Console\Input\ArrayInput

ArrayInput represents an input provided as an array.

Usage:

$input = new ArrayInput(array('name' => 'foo', '--bar' => 'foobar'));

Implements
