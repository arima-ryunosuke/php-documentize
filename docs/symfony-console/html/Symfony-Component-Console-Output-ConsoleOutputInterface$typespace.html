Structure (Symfony\Component\Console\Output\ConsoleOutputInterface)

abstract

interface Symfony\Component\Console\Output\ConsoleOutputInterface

ConsoleOutputInterface is the interface implemented by ConsoleOutput class.
This adds information about stderr output stream.

Construction

Implements
Symfony\Component\Console\Output\OutputInterface OutputInterface is the interface implemented by all Output classes.
