Structure (Symfony\Component\Console\Output\NullOutput)

class Symfony\Component\Console\Output\NullOutput

NullOutput suppresses all output.

$output = new NullOutput();

Construction

Implements
Symfony\Component\Console\Output\OutputInterface OutputInterface is the interface implemented by all Output classes.
